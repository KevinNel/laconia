# Laconia

A PHP Application.

## Instructions

0. Have Apache, MySQL, and PHP installed and running
1. Point a server root to `/public`.
2. Run the `install.php` script to create the database.
3. Run `composer install`.
4. Explore.

## Features

- Register
- Login
- Logout
- Reset password
- Create list

## Learning

- MVC
- OOP
- Routing
